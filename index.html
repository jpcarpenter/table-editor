CATS Route Editor

This is a replica of the WYSIWYG editor on the CATS website. It fixes many of the issues such as not being able to select rows/columns and copy/paste into them.

How to work it:

  1. While on a CATS routes page, find and click on "Open WYSIWYG Editor" button.
  2. Cick Tools > Source Code. highlight all code (⌘ + a) and copy (⌘ + c)
  3. Come back to this page and click Tools > Source Code. Paste the code you just copied in this window (⌘ + v).
  4. Vióla, you now have the CATS style table pasted here.
  5. 🎉🎉 Have fun highlighting and pasting your rows and columns 🎉🎉
  6. When you're done, follow the same steps in step 3 to copy your newly edited table and paste it within the CATS WYSIWYG editor.
